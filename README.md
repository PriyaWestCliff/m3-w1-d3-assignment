# m3-w1-d3-assignment (Day 3 Lab)

## Overview
Node.js + Express app that renders a registration form using Pug.  
Includes POST validation, MongoDB integration, CSS styling, and a protected route using HTTP Basic Authentication.

## Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose
- Pug
- Bootstrap 4
- Nodemon (dev)

## Run
```bash
npm install
npm run watch
